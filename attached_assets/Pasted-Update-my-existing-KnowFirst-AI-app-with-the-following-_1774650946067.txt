### 5. RICH STRUCTURED EXPLANATION RENDERER

Create a reusable React component called "ExplanationRenderer" 
that takes the JSON response from either generate-explanation or 
explain-material and renders it beautifully.

Render each field as follows:

TITLE:
- Large heading, font-weight 700, colored with the level accent color
  (green for Child, amber for Beginner, purple for Expert)
- If no level (material-only mode), use a neutral dark color

SUMMARY:
- Italic paragraph in a soft background card
- Border-left accent line in the level color

SECTIONS (loop through sections array):
- Each section has a heading (h3 style) and content rendered by type:

  type = "text":
  → Render as a paragraph with line-height 1.8

  type = "bullets":
  → Render as a styled unordered list
  → Each bullet has a small colored square or dot accent
  → Generous spacing between items

  type = "table":
  → Render as a proper HTML table
  → Striped rows (alternating background)
  → Header row with level accent color background
  → Borders: 0.5px, rounded corners on the table wrapper
  → Horizontally scrollable on small screens

  type = "keyvalue":
  → Render as a two-column definition list
  → Left column: bold key in level accent color
  → Right column: value in normal text
  → Divider line between each row

KEY TERMS:
- Section heading "Key Terms"
- Render as a grid of cards (2 columns on desktop, 1 on mobile)
- Each card: term in bold at top, definition below in smaller text
- Card has subtle border and rounded corners

KEY TAKEAWAYS:
- Section heading "Key Takeaways"
- Render as a numbered list with large colored numbers
- Each item on its own row with generous padding

COMMON MISCONCEPTIONS:
- Section heading "Common Misconceptions"
- Render each as a row with a red X icon on the left
- Background: very light red/pink tint per row

SUGGESTED QUESTIONS:
- Section heading "Want to go deeper? Try these questions"
- Render each as a clickable chip/button
- On click: call sendPrompt() with that question text so the 
  user can explore further in the chat

LEVEL BADGE:
- Show the determined level badge (Child/Beginner/Expert) at the 
  top of the explanation section
- If material-only mode, skip the badge
